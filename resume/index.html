    小型聲量分析專案

    當初初學爬蟲時，結合自己編寫的關鍵字提取演算法，配合TF-IDF演算法，達成非監督式學習的效果：爬取一段段的新聞，自動擷取關鍵字以及過濾垃圾訊息，結合文字權重計算以計算出討論聲量。

    圖中圓點越大表示在該時段具有越高的討論度。該資料為2019年八到九月份時自新聞網上爬取，所以顯而易見熱衷討論的話題是總統大選。

    除了政治分類外，尚有其他諸如娛樂、科技等等的分類標籤可供選擇（可複選），亦會顯現出該分類底下熱衷討論的話題是什麼。

    網頁連結 Github
    隨機生成圖片相簿

    此為初學jQwery時結合Bootstrap RWD做出的純前端CRUD互動靜態網頁。

    按下隨機可隨機生成圖片，當然也可以在網址列貼上圖片網址，並按下貼上即可新增到圖片預覽區。如果網址不符合格式，會跳出可愛的錯誤訊息。

    在圖片上按一下可以放大該圖片，並且在大圖底下會有返回與刪除鈕。

    網頁連結 Github
    Git job找工作網站

    此為Javascript串接Git job API來取得職缺資料的網站。使用到Axios來接收JSON資料。

    初次載入網頁時即會顯示全部職缺，左側（手機版頂端）description欄位可輸入欲搜尋條件，比如想找Ruby職缺就輸入Ruby，想要找同時會前者跟Python的職缺的話就空白間隔輸入Python。

    Location欄位可輸入想工作的地點，full time only則是僅顯示全職工作。按下search後靜待片刻職缺即會顯示在右方（手機版下方）。一次僅顯示50筆，捲到最下方可按next page以顯示更多資料。
